CSS Overflow Examples

Overflow: visible

This content is very long and will overflow the box. You can see how it extends outside the container when overflow is set to "visible". The text is now much bigger to clearly demonstrate the overflow behavior.

Overflow: hidden

This content is very long and will overflow the box. However, it is clipped because overflow is set to "hidden". The text size has been increased for better visibility.

Overflow: scroll

This content is very long and will overflow the box. Scrollbars are added to let you view the entire content. The text size has been increased to highlight the effect.

Overflow: auto

This content is very long and will overflow the box. Scrollbars are added only if necessary. The larger text helps demonstrate this behavior effectively.

Overflow-x and Overflow-y

This content is very long and will overflow the box. Here, we are controlling the overflow separately for horizontal and vertical directions using overflow-x and overflow-y. The larger text helps emphasize the effect.
